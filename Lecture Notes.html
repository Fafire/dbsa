Lecture Notes
 

Our Course has two parts:

Theory part: attaches great importance to the principle, technology and introduction of method of the basic model, the main focus is divided into three parts:

  • Relational database model
  • Object relational database model
  • Database system design technology

Practice part: pay attention to the mastery and application of actual database products, class time is thirteen hours, eight practical subject.
