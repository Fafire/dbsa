Homework-upload Instuction

1.Download github desktop at https://github-for-windows.en.softonic.com/


2.Register your GitHub account，and Log in to your GitHub account on GitHub desktop. Tell your teacher your GitHub account and then add your account to GitHub organization.


3.Right click on "view on GitHub" on the right and find organization “USST DATA"on the page.

4.creat your own repositories by using your name.Click on the "new" on the right to fill in your name in "Repository name" and the warehouse type is "public" and then click "creat repository".


5.Open your github desktop.Click + number,then clicking "clone" to clone your project and chosing where to save.Then clone it.


6.Open the project you have downloaded and find the file which you have cloned and put your homework in it.


7.Open your github desktop.There is a white dot next to changes that the project file has been changed and that your work has been put in.Fill the summary with your name,same as description,at last click "commit to master"button.


8.Click "Sync"or"publish" to upload updates to the server to complete the job submission
